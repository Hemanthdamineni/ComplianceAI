What are the key changes in the updated ISO 27001:2022 standard regarding cloud security controls compared to the 2013 version?

robot_2

The transition from ISO/IEC 27001:2013 to the 2022 revision introduces significant structural and content updates, particularly concerning cloud environments.

Key changes include:

  • Consolidation of Controls: The total number of controls decreased from 114 to 93.
  • New Cloud-Specific Control: Control 5.23 "Information security for use of cloud services" was explicitly added.
  • Threat Intelligence: A new control (5.7) focusing on gathering and analyzing threat intelligence has implications for cloud deployments.
